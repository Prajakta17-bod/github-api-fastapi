# Github API Fastapi Project

## Overview ##

This is a FastAPI-based backend project that interacts with the GitHub API.
It allows users to fetch repositories and issues of a GitHub user through simple API endpoints.

**Tech Stack**

Python

FastAPI

Uvicorn

Request


 ### Setup Instructions ##3
**1. Clone the repository**

git clone https://github.com/Prajakta17-bod/github-api-fastapi.git

**2. Navigate to project directory**

cd github-api-fastapi

**3. Create virtual environment**

python -m venv venv
